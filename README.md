# Zen Mods

Collection of Sine mods for Zen Browser.

## About

This repository contains various mods for Zen Browser that can be installed via [Sine](https://github.com/CosmoCreeper/Sine), the ultimate theme manager for Firefox-based browsers.

## Installation

### Prerequisites

1. Install [Sine](https://github.com/CosmoCreeper/Sine) for Zen Browser
2. Open Zen Browser and go to Settings → Sine Mods
3. Enable "Download JS from unofficial sources" in Sine settings

### Installing Mods

1. In Sine Mods settings, below the marketplace, paste this repository link:
   ```
   https://github.com/antajl/zen-mods
   ```
2. The mods should install automatically
3. Restart Zen Browser if prompted

## Available Mods

### Always Visible Bookmarks Bar

Disables hover-to-show behavior for bookmarks bar in Compact Mode. Makes the bookmarks bar always visible instead of hiding until hover.

**Features:**
- Always visible bookmarks bar
- Compact Mode fix
- Minimal CSS changes
- No layout modifications

See [always-visible-bookmarks/](always-visible-bookmarks/) for details.
